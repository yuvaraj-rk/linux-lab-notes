# Useful Shell Commands
---

### To browse files from terminal effectively
```shell
ranger
```

### To quickly find common command usage:
```shell
tldr <command name>
#eg:
tldr find
```

### Search for file path in a directory quickly:
```shell
fzf <filename>
```

### Modern cat command with syntax highlighting
```shell
batcat
```

### Search for a keyword ripgrep (rg) in a file
```shell
rg <keyword in a file inside a directory>
```

---

## Important Notes
* **Installation Required:** Most advanced utilities mentioned here are **not** available by default. You must install them manually (e.g., `sudo apt install` or `brew install`).
* **Environment:** All commands and scripts have been tested and verified using **WSL2 (Windows Subsystem for Linux)**.
* **System Tools:** These exercises focus on enhancing productivity, system monitoring, and automation.
