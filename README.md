#To browse files from terminal effectively
```shell
ranger
```

# To quickly find common command usage:
```shell
tldr <command name>
#eg:
tldr find
```

#Search for file path in a directory quickly:
```shell
fzf <filename>
```

#modern cat command with syntax highlighting
```shell
batcat
```

#search for a keyword ripgrep (rg) in a file
```shell
rg <keyword in a file inside a directory>
```
